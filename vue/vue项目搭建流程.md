### 搭建vue2.x项目流程

1. 全局安装vue-cli

   ```
   npm i vue-cli -g
   ```
2. 创建一个项目
   ```
   vue init webpack hello_world
   ```

3. 安装依赖
   ```
   npm i
   ```

### 搭建vue3.x项目流程

说明：Vue CLI 的包名称由 `vue-cli` 改成了 `@vue/cli`

1. 全局安装vue-cli

   ```
   npm i @vue/cli -g
   ```

2. 创建一个项目

   ```
   vue create hello_world
   ```
