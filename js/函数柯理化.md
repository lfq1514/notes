###  js函数柯理化

######  概念:

     ######   就是把接受多个参数的函数，变成接受单一参数的函数，并且返回接受剩余参数且返回结果是新函数的技术

```
curry(fn){
    let len=fn.length
    return temp(){
    //收集传递进来的参数
        let args=[...arguments]
        //如果传进来args的参数>=fn函数中的参数，就将参数传给fn并去立即执行。 否则就不去执行
        if(args.length>=len){
        return fn(...args)
        }else {
            return function(){
                return  temp(...args,...arguments)
            }
        }
    }
}
```
