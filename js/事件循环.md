# 事件循环

同步任务-->微任务-->浏览器渲染-->宏任务-->同步任务---

宏任务每个循环只执行一个
微任务每次循环都会清空，微任务执行期间产生的微任务会在本轮执行完

await==>promise.resolve().then(await后面要执行的代码)

click事件只有手动点击触发，才算是宏任务

requestAnimationFrame,requestIdleCallback 可以理解为是和浏览器渲染相关的钩子，（严格意思上讲不是微任务，是在微任务之后执行的）
